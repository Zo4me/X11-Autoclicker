# X-11 Autoclicker

## IMPORTANT INFORMATION:
This autoclicker is **LINUX** only. This autoclicker is only compatible with the **Xorg (X11)** compositor.

## **options:**
-h, --help         show this help message and exit
--cps CPS          Target clicks per second (default: 10.0)
--jitter JITTER    Jitter around the interval, in milliseconds, applied as +/- this amount (default: 3.0)
--device DEVICE    Exact device name to match. If flag is prompted, you must add the exact device name.
--trigger TRIGGER  Button that arms/disarms clicking (default: BTN_EXTRA)
--button BUTTON    Button that gets synthetically clicked (default: BTN_LEFT)
--list-devices     List connected input devices and their recognized button capabilities, then exit

### The autoclicker automatically sets CPS to 10, jitter to 3, and auto-detects your device. It will assume your mouse button.
